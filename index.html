Faith's Projects

My personal projects are done in a variety of languages, including HTML/CSS/JavaScript, Java, and Python.

Habit Tracker Website

This website built in HTML/CSS/JavaScript leverages form functionality, web storage, and jQuery so users can add, delete and check off the daily habits they are trying to build each month. It was inspired by James Clear's book Atomic Habits.

Brain Freeze

Brain Freeze is an app with a Java Swing GUI for users to study with various active-recall techniques. In the flashcards, the program manipulates Array and HashMap data structures to store and display flashcard sets. For blurting, the program analyzes the original text given by the user and what they submit from memory, providing an accuracy score based on corresponding words between the texts. With the first letter technique, it scans through a given text and displays the first letter of each word for memorization.

Mood Tracker

The Mood Tracker app has three features: mood tracking, reflection prompts, and mood regulation exercises. In mood tracking, users can input numerical data on how they feel, which is written to a file and stored, then graphing the trend in Java Swing. The reflection uses text input and prompt randomization for users to reflect on a given mood-related prompt. The mood regulation feature is a form that suggests an exercise based on the user's answers.

Grade Calculator

This Python grade calculator takes users' input grades and stores the assessment name and grade in a list. It calculates a weighted average and can take grades in percentages, fractions, letters, or levels from 1 to 4. It can also display all input assessments and their corresponding grades.
